Xfields package documentation
=============================



Xfields is a python package beloging to the xsuite library for the simulation of beam dynamics in particlerators. Xfields in particular allows computing a the electromagnetic fields generated by particle ensambles using Particle In Cell (PIC) solvers or simplified distributions.
